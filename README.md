# iPhone単体 台判定アプリ

PCサーバーを使わず、iPhone内に保存した台履歴だけで計算する軽量版です。

## できること

- CSV/JSONをiPhoneで取り込む
- 取り込んだデータをiPhone内に保存する
- 台番、現在回転、見る範囲を指定して即時計算する
- 確変込みの当たり割合、初当りのみ、信憑性、最近傾向、リスクリターンを見る
- インストール後はオフラインで起動する

## インストール

iPhoneにホーム画面アプリとして入れるには、初回だけHTTPSでこのフォルダを公開してSafariで開きます。

1. `iphone-standalone` フォルダをGitHub Pages、Netlify、Cloudflare Pagesなどの静的ホスティングに置く
2. iPhoneのSafariで公開URLを開く
3. 共有ボタンを押す
4. `ホーム画面に追加` を選ぶ
5. 追加された `台判定` アイコンから起動する

一度開けば、アプリ本体はiPhoneにキャッシュされます。取り込んだデータもiPhone内のIndexedDBに保存されるため、PCの電源は不要です。

## データ取り込み

アプリの `データ` 画面からD-Deltaの台履歴CSVを選んで `取り込む` を押します。

PC側のパチスロ分析アプリを使っている場合は、PC側の `データ` 画面にある `iPhone転送` からQRコードを作成できます。iPhoneでQRを読み取り、`pachislo-lite-data.json` を保存してから、このアプリで取り込んでください。PCは転送時だけ起動していればよく、取り込み後は不要です。

最低限必要な列:

- `machine_name`: 機種
- `unit_label`: 台番
- `history_date` または `history_day`: 日付
- `スタート`: 回転数

あると精度が上がる列:

- `is_chance_hit`: 確変・ST・RUSH中の当たり判定
- `大当り回数`: 当たり番号
- `時間`: 当たり時刻
- `獲得数（継続数）`: 出玉情報

## 注意

これは保存済みの当たり履歴から分布を計算する道具です。未当選で終わった回転はCSVに入らないため、完全な期待値計算ではありません。画面の信憑性表示で、到達サンプル数、履歴日数、確変判定率を確認してください。
